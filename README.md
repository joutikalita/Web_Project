MovieHub and FashionHub
Overview

This repository contains two frontend web projects created to practice web development and UI design. Both projects focus on building responsive layouts and clean user interfaces using basic web technologies.

The projects included are:

MovieHub – A movie browsing website
FashionHub – A simple online shopping interface
MovieHub

MovieHub is a movie listing website designed to display movies in a structured and visually appealing way.

Features
Grid-based layout for movies
Movie cards with images and titles
Responsive design for different screen sizes
Simple and clean dark-themed interface
Purpose

The project helps in understanding layout design, card components, and basic interactivity.

FashionHub

FashionHub is an online shopping frontend that showcases fashion products in a modern layout.

Features
Product cards with images, price, and details
Responsive design
Navigation bar and sections
Clean and organized UI
Purpose

This project focuses on designing an e-commerce style interface and improving UI consistency.

Technologies Used
HTML for structure
CSS for styling and layout
JavaScript for basic interactivity
